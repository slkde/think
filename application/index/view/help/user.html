账户绑定

1.点击头像右上角的个人头像
2.点击设置按钮
3.先绑定手机
4.输入手机号后点击获取验证码，待验证码发送到手机后输入验证码点击绑定即可
5.继续点击 绑定SteamID
6.输入您Steam的账号和密码点击登陆
7.绑定成功后，网页跳出提示窗口
8.最后，来绑定交易链接 点击“点击获取交易URL“
9.将“第三方网站”下“交易URL”下的链接 复制
10.粘贴到{:config('WEN_NAME')}的用户信息中 点击保存，完成。
